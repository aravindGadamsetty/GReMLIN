  • Do we need separate license for JChem .NET API.

    No. JChem license for Java is also valid for .NET usage.

  • Where are the other applications form the JChem package?

    JChem .NET is only an API, so it does not contain any applications, which are present in the Java package.

  • How can I import a structure from InChi?

    The same way as it is in Java, plus JChemExt.dll should be referenced and ChemAxon.JChem.Common.Init() should be called before using InChi.
